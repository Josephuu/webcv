# Joseph Gavidia - CV Web

Sitio web de currículum vitae personal de **Joseph Gavidia**, Cloud DevOps Engineer.

## Descripción

Este es un sitio web estático que presenta el perfil profesional, experiencia laboral, habilidades técnicas y proyectos destacados de Joseph Gavidia. Incluye animaciones interactivas, efectos de partículas, modo oscuro/claro y diseño responsive.

## Tecnologías

- HTML5
- CSS3 (variables CSS, animaciones, grid/flexbox)
- JavaScript vanilla (Intersection Observer, Canvas API, LocalStorage)

## Estructura

| Archivo | Descripción |
|---------|-------------|
| `index.html` | Estructura principal del sitio |
| `styles.css` | Estilos, temas oscuro/claro y animaciones |
| `script.js` | Interactividad, partículas, typing effect y theme toggle |
| `FotoCv.jpg` | Foto de perfil |

## Despliegue

Este sitio está configurado para desplegarse en **Vercel** como proyecto estático.

### Pasos para desplegar:

1. Sube la carpeta `webcv/` a un repositorio de GitHub.
2. Importa el repositorio en [vercel.com](https://vercel.com).
3. Vercel detectará automáticamente la configuración de `vercel.json`.
4. El sitio se desplegará y estará disponible en una URL pública.

## Notas

- Las rutas de recursos son relativas para garantizar compatibilidad en producción.
- El botón "Descargar CV" requiere que el archivo DOCX esté disponible en la misma carpeta o en una ruta accesible.
